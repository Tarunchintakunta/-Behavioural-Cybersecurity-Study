\section{System Requirements}
\label{sec:sys_req}

This section outlines the hardware and software requirements necessary to run the Phishing Behavioral Cybersecurity Study framework.

\subsection{Hardware Requirements}
\begin{itemize}
    \item \textbf{Processor}: Intel Core i5 or equivalent (Minimum), Intel Core i7 or higher (Recommended).
    \item \textbf{RAM}: 8 GB (Minimum), 16 GB (Recommended) for efficient machine learning model training.
    \item \textbf{Storage}: At least 2 GB of free disk space for the project files, virtual environment, and datasets.
\end{itemize}

\subsection{Software Requirements}
\begin{itemize}
    \item \textbf{Operating System}: Windows 10/11, macOS (10.15+), or Linux (Ubuntu 20.04+).
    \item \textbf{Python}: Version 3.9 or higher.
    \item \textbf{IDE}: Visual Studio Code (Recommended) or PyCharm.
    \item \textbf{Terminal}: Bash, Zsh, or PowerShell.
\end{itemize}

\subsection{Python Dependencies}
The project relies on the following key Python libraries, which are listed in \texttt{requirements.txt}:
\begin{itemize}
    \item \textbf{pandas}: For data manipulation and analysis.
    \item \textbf{scikit-learn}: For machine learning model training and evaluation.
    \item \textbf{streamlit}: For the interactive dashboard.
    \item \textbf{mlflow}: For experiment tracking.
    \item \textbf{statsmodels} \& \textbf{pingouin}: For statistical analysis.
    \item \textbf{matplotlib} \& \textbf{seaborn} \& \textbf{plotly}: For data visualization.
\end{itemize}
