\section{System Requirements}
\label{sec:sys_req}

This section outlines the hardware and software requirements necessary to run the Phishing Behavioral Cybersecurity Study framework.

\subsection{Hardware Requirements}
\begin{itemize}
    \item \textbf{Processor}: Intel Core i5 or equivalent (Minimum), Intel Core i7 or higher (Recommended).
    \item \textbf{RAM}: 8 GB (Minimum), 16 GB (Recommended) for efficient machine learning model training.
    \item \textbf{Storage}: At least 2 GB of free disk space for the project files, virtual environment, and datasets.
\end{itemize}

\subsection{Software Requirements}
\begin{itemize}
    \item \textbf{Operating System}: Windows 10/11, macOS (10.15+), or Linux (Ubuntu 20.04+).
    \item \textbf{Python}: Version 3.9 or higher.
    \item \textbf{IDE}: Visual Studio Code (Recommended) or PyCharm.
    \item \textbf{Terminal}: Bash, Zsh, or PowerShell.
\end{itemize}

\subsection{Python Dependencies}
The project relies on the following key Python libraries, which are listed in \texttt{requirements.txt}:
\begin{itemize}
    \item \textbf{pandas}: For data manipulation and analysis.
    \item \textbf{scikit-learn}: For machine learning model training and evaluation.
    \item \textbf{streamlit}: For the interactive dashboard.
    \item \textbf{mlflow}: For experiment tracking.
    \item \textbf{statsmodels} \& \textbf{pingouin}: For statistical analysis.
    \item \textbf{matplotlib} \& \textbf{seaborn} \& \textbf{plotly}: For data visualization.
\end{itemize}

\subsection{Project Structure}
The project follows a modular directory structure to separate data, source code, and results.

\begin{itemize}
    \item \texttt{config/}: Contains configuration files (e.g., \texttt{config.yaml}) for model parameters.
    \item \texttt{data/}: Stores all datasets.
    \begin{itemize}
        \item \texttt{raw/}: Original survey and experiment data.
        \item \texttt{processed/}: Cleaned data ready for modeling.
        \item \texttt{synthetic/}: Generated data for testing purposes.
    \end{itemize}
    \item \texttt{mlruns/}: MLflow tracking data (metrics, parameters, artifacts).
    \item \texttt{models/}: Saved trained models (e.g., \texttt{.joblib} files).
    \item \texttt{results/}: Output reports from statistical analysis.
    \item \texttt{src/}: Source code for the project.
    \begin{itemize}
        \item \texttt{analysis/}: Scripts for statistical testing.
        \item \texttt{data\_collection/}: Scripts for data ingestion and cleaning.
        \item \texttt{models/}: Machine learning training and evaluation scripts.
        \item \texttt{processing/}: Data preprocessing pipelines.
        \item \texttt{utils/}: Helper functions and logging utilities.
        \item \texttt{visualization/}: Streamlit dashboard code.
    \end{itemize}
    \item \texttt{tests/}: Unit tests for the codebase.
    \item \texttt{requirements.txt}: List of Python dependencies.
\end{itemize}
